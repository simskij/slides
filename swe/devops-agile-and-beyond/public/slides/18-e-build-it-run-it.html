BUILD IT & RUN IT

  • En av de första grejerna vi gjorde när vi byggde vårt nya utvecklingskontor på Axis, var att bestämma oss för att inte ha något ops-team.
  • Med det inte sagt att vi skulle vara helt utan uppgifter som normalt utförs av Ops.
  • Däremot konstaterade vi att separationen av ops och devs snarare leder till att dev helt saknar incitament för att säkerställa att deras produkter är lätta att drifta och kräver så lite underhåll som möjligt. Genom att teamen själva tar ansvar för att drifta sina produkter, säkerställer vi alltså att teamen bygger bort arbetsuppgifter de inte tycker är givande att utföra.
